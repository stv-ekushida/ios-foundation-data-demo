# Data

## 概要
Dataはバイトデータのラッパークラスです。

## 主要プロパティ

| プロパティ名 | 説明 | サンプル |
|-----------|------------|------------|
| count | バイト数を表すInt | data.count |
| isEmpty | 空かどうか | data.isEmpty |

## 主要メソッド

| メソッド名 | 説明 | サンプル |
|-----------|------------|------------|
| base64EncodedData() | base64エンコードする | data.base64EncodedData() |

## フレームワーク
Foundation.framework

| Category | Version |
|:-----------:|:------------:|
